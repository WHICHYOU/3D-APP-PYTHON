# SDK Documentation

## 2D to 3D Conversion SDK

This SDK allows hardware manufacturers and developers to integrate 2D to 3D conversion capabilities into their products.

## Target Partners

- VR/AR Headset Manufacturers (Meta, Apple, XREAL)
- Display Manufacturers (Samsung, LG, Sony)
- Video Streaming Platforms
- Game Engines

## Integration Modes

### 1. Python SDK

Direct Python integration for maximum flexibility.

### 2. C++ SDK (Coming Soon)

High-performance C++ wrapper for embedded systems.

### 3. REST API (Coming Soon)

Cloud-based processing for web applications.

## Features

- Real-time depth estimation
- Stereoscopic rendering
- Multiple output formats (SBS, Top-Bottom, Anaglyph)
- Hardware acceleration (CUDA, Metal, OpenCL)
- Batch processing
